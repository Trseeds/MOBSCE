SpriteRenderParameters

Members
SDL_Texture* Texture - Texture that will be rendered, set automatically.
Vector3 Position - Position the sprite will be rendered at.
Vector4 Origin - The rectangular area within the source texture that will be visivble when rendered.
Vector2 Dimensions - The dimensions of the texture when rendered.
int Transparency - 0-100 value that controls how see-through the texture will be when rendered.
double Angle - 0-360 value that controls the rotation in degrees of the texture will be when rendered.
int Flip - Controls if the texture will be mirrored when rendered. Use enums FLIP_*.
Vector3 Tint - Color tint applied to texture when rendered. X RED, Y GREEN, Z BLUE.
int Visible - The sprite will skip rendering if this flag is not set.
Description
This struct is used during rendering only, and controls various effects and parameters relating to rendering.
