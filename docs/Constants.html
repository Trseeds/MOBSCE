Constants

Engine Constants

STRING_BUFFER_SIZE

Value
int - 1024
Usage
For string operations. Use when declaring strings for GetAssetPath.

OBJECT_NAME_SIZE

Value
int - 64
Usage
Sprite and actors need names, but should not be bloated with a 1k string.

MIN_ALLOCATE

Value
int - 16
Usage
This many objects are allocated (or deallocated) in resource functions.

EVENT_QUEUE_SIZE

Value
int - 512
Usage
SDL events can overflow and drop events (espescially with a gamepad connected) so we store this many events.

INVALID_ENGINE

Value
int - -1
Usage
Return value.
