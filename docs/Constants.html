Engine Constants

Bool

stdbool is included, use true and false instead of 1 and 0 for clarity.

STRING_BUFFER_SIZE

Value
int - 1024
Usage
For string operations. Use when declaring strings for GetAssetPath.

OBJECT_NAME_SIZE

Value
int - 64
Usage
Sprite and actors need names, but should not be bloated with a 1k string.

MIN_ALLOCATE

Value
int - 16
Usage
This many objects are allocated (or deallocated) in resource functions.

EVENT_QUEUE_SIZE

Value
int - 512
Usage
SDL events can overflow and drop events (espescially with a gamepad connected) so we store this many events.

INVALID_ENGINE

Value
int - -1
Usage
Return value.

Video

TINT_NOCHANGE

Value
int - 255
Usage
Used to not change the value of a color channel in renderering.

FLIP_*

Usage
Used to control the flip of a sprite during rendering.

Input

MB_*

Usage
Index for mouse button array.

SCROLL_*

Usage
Comparison for mouse scroll checking.

K_*

Usage
Index for the keyboard array.

GP_*

Usage
Index for gamepad arrays.
Note
The enums are named ambiguously intentionally. The mappings to real gamempads are as follows:
MOBSCE - RIGHT BOTTOM LEFT TOP START SELECT SUPER
Nintendo (Wii Classic Controller+) - A B Y X PLUS MINUS HOME
Xbox (One+) - B A X Y MENU VIEW XBOX
Xbox/Generic - B A X Y START BACK GUIDE
Sony (PS4+) - CIRCLE CROSS SQUARE TRIANGLE OPTIONS SHARE PS
Sony (PSX-PS3) - CIRCLE CROSS SQUARE TRIANGLE START SELECT PS
